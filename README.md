# Maternal Health App Risk Assessment

This project developed a data-driven framework to evaluate pregnancy and maternal health apps that claim to identify high-risk pregnancies.

## Project Overview

As part of a 4-member University of Maryland TLP team, we analyzed publicly available maternal health datasets, performed data cleaning and wrangling, and developed Python-based machine learning models to identify key pregnancy risk indicators.

## Key Features

- Cleaned and prepared maternal health datasets
- Built and compared machine learning models using Python
- Identified important risk factors such as blood sugar and gestational diabetes
- Designed a rubric-style framework for evaluating maternal health apps
- Created a project website to present the findings
